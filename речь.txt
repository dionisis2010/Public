Почему мы выбрали следующие технологии:
java - объектно-ориентированный, высокопроизводительный, переносимый язык с встроенной поддержкой работы в сетях.

framework Spring - возможность разработки приложения как набора слабосвязанных (loose-coupled) компонентов
    представляет собой набор фреймворков для решения различных задач

postgreSQL - надежная система управления базам данных,
 имеет хорошие характеристики по производительности, работает практически на всех UNIX-платформах

hibernate - Библиотека дающая удобный Апи для работы с базами данных,
    позволяет представлять записи из базы данных как объекты в java коде, работает на основе JDBC драйвера

maven -  фреймворк для автоматизации сборки проектов на основе описания их структуры

git - удобная распределенная система контроля версий,
    позволяет нескольким разработчикам удаленно работать над одним проектом

UI - гибкость, быстрый и наглядный результат своей работы
    html - сохранение информации шаблона, открытый стандарт
    css - улучшенные стили, оффлайн-просмотр, независимость от платформы
    js - гибкость языка, возможность использования множества шаблонов программирования
    bootstrap.min.css - минимизированый файл для быстрой загрузки сайта
    шаблонизатор Таймлиф - позволяет в рантайме генерировать страницы из заготовок страниц, ресурсов и данных из программы
